nona
GitHub

Rename files,
your way.

A fast, zero-dependency CLI for normalizing filenames into kebab-case, snake_case, or CamelCase.

$ brew tap sanojsubran/apps && brew install nona

Features

⚡

Fast & lightweight

Single binary, no runtime dependencies, written in Go.

🎨

Multiple styles

Rename to kebab-case, snake_case, or CamelCase with one flag.

🛡

Safe by default

Files whose names are already normalized are left untouched.

📦

Batch renaming

Pass any number of files in a single command.

Usage

shell
nona [--style kebab|snake|camel] <file> [file ...]
examples
# Default: kebab-case
$ nona "My Screenshot.png"
My Screenshot.png -> my-screenshot.png

# snake_case
$ nona --style snake "Hello World.txt"
Hello World.txt -> hello_world.txt

# CamelCase
$ nona --style camel "hello world.txt"
hello world.txt -> HelloWorld.txt

# Batch rename
$ nona *.png

Styles

Flag value Input Output
kebab default Hello World.txt hello-world.txt
snake Hello World.txt hello_world.txt
camel Hello World.txt HelloWorld.txt

Install

homebrew 
$ brew tap sanojsubran/apps
$ brew install nona
go install 
$ go install github.com/sanojsubran/nona/cmd/nona@latest
build from source 
$ git clone https://github.com/sanojsubran/nona
$ cd nona
$ make build
